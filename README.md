## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- Python >= 3.8
- Docker >= 20.0.0

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/username/repo.git

# 进入项目目录
cd repo

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### Docker 部署

```bash
# 构建镜像
docker build -t your-project .

# 运行容器
docker run -p 3000:3000 your-project
```
